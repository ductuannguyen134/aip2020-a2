# Dev Start commands:
Frontend dev start:
- cd frontend
- npm start


Backend dev start:
- cd backend
- npm start
(if nodemon is installed, you can use nodemon server.js instead)

# Build:
- npm run build

# Deploy:
- npm run deploy
